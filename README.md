**SYSTEM DESCRIPTION**

Asset Manager is a comprehensive web-based inventory and asset management system designed to help organizations efficiently track, manage, assign, and maintain both IT and non-IT assets. Built using PHP and MySQL, the system provides secure role-based access control, configurable approval workflows, real-time tracking, and advanced reporting tools to ensure accountability, transparency, and optimal asset utilization.

**1. Authentication & Access Control**
Secure user authentication with password hashing
Role-based access: Admin, Manager, Supervisor, User
Session management with automatic timeout
User registration with admin approval

Activity logging with IP tracking
